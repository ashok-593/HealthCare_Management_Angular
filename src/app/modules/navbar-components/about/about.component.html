About SmartCare Health System

SmartCare is an end-to-end appointment and consultation management system built for modern healthcare needs. It simplifies the communication between patients and doctors while ensuring secure and timely medical services.

Key Objectives:

  • Enable patients to book and manage appointments 24/7
  • Provide doctors a centralized dashboard to manage consultations
  • Ensure real-time availability and notifications
  • Maintain secure and reliable patient records

Technology Stack:

  • Backend: Java + Spring Boot + JPA + MySQL
  • Frontend: Angular + Bootstrap
  • Security: JWT-based authentication
  • Deployment: Google Cloud Platform (GCP)

This project is designed for scalability, with modular features such as notifications, consultations, and more. We're committed to improving digital healthcare, one line of code at a time.
